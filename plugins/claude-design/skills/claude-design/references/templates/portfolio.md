# Template: Portfolio

업종: 개인 포트폴리오 / 에이전시 / 프리랜서
목적: 작업물 소개, 고용/연락 유도
권장 스타일 태그: `editorial, minimal, creative`

---

## 섹션 구조

```
① Navbar          — Logo/이름 + 메뉴 (About, Work, Contact) + 언어 전환(선택)
② Hero            — 이름 + 타이틀 + 한 줄 소개 + CTA (작업 보기 / 연락하기)
③ About           — 사진 + 소개 텍스트 + 스킬 배지 or 연도별 경력
④ Work            — 프로젝트 그리드 (마소리 or 균등 그리드)
⑤ Skills          — 기술 스택 (아이콘 + 이름) 또는 카테고리별 그룹
⑥ Testimonials    — 클라이언트 후기 2-3개 (선택)
⑦ Contact         — 이메일 + SNS 링크 + 연락 폼 (선택)
⑧ Footer          — 저작권 + SNS 아이콘
```

---

## 레이아웃 패턴

### Hero
```
[전체 화면 높이 또는 80vh]
  중앙 정렬 or 좌측 정렬:
    안녕하세요, 저는
    [이름] — 타이포 대형 (60-96px)
    [직함/전문 분야]
    [한 줄 소개]
    [작업 보기 →]  [연락하기]
```

### Work 그리드 (마소리)
```
[ 프로젝트1  ][ 프로젝트2 ]
[ 프로젝트3  ——————————— ]
[ 프로젝트4 ][ 프로젝트5 ]

각 카드: 썸네일 이미지 + hover → 프로젝트명 + 카테고리 오버레이
클릭 → 모달 또는 개별 페이지
```

### About 섹션
```
좌 40%: 프로필 사진 (약간 기울임 or 컨테이너 장식)
우 60%: 소개 텍스트 (2-3단락)
        [스킬 태그 그룹]
        경력 연도 타임라인 (선택)
```

---

## 콘텐츠 가이드라인

- **Hero 이름**: 최대한 크게 (타이포가 비주얼)
- **Work 썸네일**: 16:9 또는 4:3 비율 유지
- **섹션 간격**: 120-160px (여백을 충분히)
- **컬러 포인트**: 1개 accent 색상만 (링크, hover, CTA)
- **애니메이션**: 스크롤 fade-in, Work 카드 hover scale(1.03)

## 브레이크포인트

```css
/* Mobile */  → 1열 그리드, Navbar → 햄버거 메뉴
/* Tablet */  → 2열 그리드
/* Desktop */ → 마소리 or 3열 그리드
```
