# Template: Newsletter (HTML Email)

업종: 뉴스레터 / 이메일 마케팅 / 미디어
목적: 콘텐츠 전달, 클릭 유도, 구독자 유지
권장 스타일 태그: `editorial, minimal, professional`

---

## 섹션 구조

```
① Preheader       — 이메일 클라이언트 미리보기 텍스트 (숨김, 50자)
② Header          — 로고 + 발행일 + 이슈 번호
③ Hero Article    — 이번 호 핵심 기사 (이미지 + 헤드라인 + 요약 + 읽기 버튼)
④ Curated List    — 3열 큐레이션 카드 (썸네일 + 제목 + 출처)
⑤ Feature Spot    — 특집 콘텐츠 (풀 너비 배너형)
⑥ Quick Links     — 짧은 링크 목록 (불릿 or 넘버)
⑦ Footer          — 구독 해지 링크 + 발신자 정보 + SNS
```

---

## 레이아웃 패턴

### 이메일 전용 규칙
```
- max-width: 600px (이메일 표준)
- 인라인 스타일 필수 (CSS class 지원 불안정)
- 이미지: alt 텍스트 필수
- 폰트: system-ui 또는 웹안전폰트 (Georgia, Arial)
- 배경: 흰색 또는 아주 밝은 회색
- 링크 색상: 명시적 지정 (기본 파란색 덮어쓰기)
```

### Header
```
[로고 중앙 or 좌측 정렬]
[뉴스레터 이름 — Issue #00 | YYYY.MM.DD]
[구분선]
```

### Hero Article
```
[썸네일 이미지, 600px 너비]
[카테고리 태그 — 작게, accent 색]
[헤드라인 — H2, 굵게, 28-32px]
[요약 2-3줄]
[읽으러 가기 →] (텍스트 링크 or 버튼)
```

### 3열 큐레이션
```
[ 카드1 ][ 카드2 ][ 카드3 ]
  이미지    이미지    이미지
  제목      제목      제목
  출처      출처      출처
  읽기      읽기      읽기

모바일: 1열 스택 (max-width 400px 이하)
```

---

## 콘텐츠 가이드라인

- **제목**: 구체적, 숫자 포함 ("왜 GPT-4는 틀렸나" > "AI 이야기")
- **요약**: 3줄 이내, 궁금증 유발
- **CTA**: "읽어보기", "자세히 보기" (일반적), 특이하게 만들지 않음
- **이미지**: 모두 alt 텍스트 필수 (이미지 차단 환경 대비)
- **길이**: 읽는 데 3-5분 (스크롤 과도하게 길지 않게)

## 테스트 체크리스트

```
□ Gmail / Outlook / Apple Mail 렌더링 확인
□ 다크모드 확인 (배경색 강제 지정)
□ 이미지 차단 시 alt 텍스트 표시
□ 모바일(375px) 1열 확인
□ 구독 해지 링크 작동
□ 모든 링크 UTM 파라미터 포함
```
