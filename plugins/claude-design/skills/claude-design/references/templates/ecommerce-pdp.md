# Template: E-commerce Product Detail Page (PDP)

업종: 이커머스 / 리테일 / D2C 브랜드
목적: 상품 정보 제공, 구매 전환
권장 스타일 태그: `professional, minimal, luxurious, b2c`

---

## 섹션 구조

```
① Navbar          — Logo + 카테고리 메뉴 + 검색 + 장바구니 아이콘
② Breadcrumb      — 홈 > 카테고리 > 상품명 (텍스트, 작게)
③ Product Main    — 이미지 갤러리(좌) + 상품 정보(우) 2열
④ Description     — 상품 상세 설명 + 스펙 테이블
⑤ Reviews         — 별점 요약 + 후기 목록 (페이지네이션)
⑥ Related         — 관련 상품 카드 4-6개 (가로 스크롤 or 그리드)
⑦ Footer          — 링크 + 저작권 + 결제 수단 아이콘
```

---

## 레이아웃 패턴

### Product Main (핵심)
```
[Breadcrumb 한 줄]

좌 55%: 이미지 갤러리
  - 메인 이미지 (크게)
  - 썸네일 스트립 (하단 or 좌측)
  - zoom-on-hover 또는 lightbox

우 45%: 상품 정보
  - 브랜드명 (작게, muted)
  - 상품명 (H1, 굵게)
  - 별점 + 리뷰 수 (링크)
  - 가격 (크게) / 할인가 + 원가 strike
  - 옵션 선택 (컬러 스와치 / 사이즈 버튼)
  - 수량 선택
  - [장바구니 담기] (primary, 전체 너비)
  - [바로 구매] (secondary or outline)
  - 배송/반품 정보 (아이콘 + 텍스트, 작게)
  - 공유 버튼
```

## 콘텐츠 가이드라인

- **이미지**: 정사각형(1:1) 또는 4:3, 최소 600px
- **상품명**: 2줄 이내
- **가격**: 가장 큰 글자 중 하나 (28-36px)
- **CTA 버튼**: 전체 너비, 충분한 높이(52-56px), 터치 타겟 보장
- **옵션 버튼**: 선택 상태 명확히 (border + background 변화)

## 브레이크포인트

```css
/* Mobile */  → 이미지 단독 → 스크롤 후 상품 정보 (1열 스택)
/* Tablet */  → 2열 (이미지 50% / 정보 50%)
/* Desktop */ → 2열 (이미지 55% / 정보 45%), max-width 1280px
```
